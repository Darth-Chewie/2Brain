* Permite organizar la información dentro de los dispositivos de almacenamiento secundario en un formato inteligible para el [[SO]]
* Se crean sobre [[particiones]] o [[volúmenes]]
* Una vez creadas las particiones, el [[SO]] debe crear las estructuras de los [[Sistema de ficheros|SSFF]] dentro de esas particiones
	* Se proporcionan comandos como format o mkfs al usuario
